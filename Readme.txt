Intructions:
1. Install python 3.9: https://www.python.org
Remember to tick the option: Add Python to path

2. Open cmd in this directory. Run:
pip install -r requirements.txt 

3. Usage:
Place the generator script in the same folder as your .html file.
Run:
generator.py <filename>
For example: 
generator.py "A- A cada instante.html"

4. Note: file name must be in this form:

[Tone]-[Name of song].html
For example:
A- A cada instante.html
